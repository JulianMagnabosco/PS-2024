# Practica Supervisada - COMOLoHAGO

Esta es una pagina web creada como tesis para la practica supervisada de la carrera de Tecnico Universitario en Programacion. Consiste en una plataforma para publicar obras de todo tipo, con imagenes, videos de Youtube, y descripcion de los pasaso para fabricarlas. Tambien permite compra, venta, y posee un sistema de delivery integrado para los repartidores.

**Se usaron:**
- Angular 17
- Bootstrap 5
- Spring Boot
- Mercado Pago API
- PostgreSQL
